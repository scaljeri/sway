Sway.data.Relation Class

Defined in: src/data/relation.js:20

A Relation defines the relation between Sway.data.Models. The following relations types are available

  • HAS_MANY
  • HAS_ONE
  • BELONGS_TO
  • HAS_MANY
  • HAS_AND_BELONGS_TO_MANY
