Sway.DI Class

Defined in: src/di.js:4

DI makes classes accessible by a contract. Instances are created when requested and dependencies are injected into the constructor, facilitating lazy initialization and loose coupling between classes.

Defined in src/di.js:79

Returns a new instance of the class matched by the contract. If the contract does not exists an error is thrown. If one of the dependencies does not exists, 'null' is used instead.

Parameters:

  • contract String
    • the contract name
  • dependencies Array
    • list of contracts passed to the constructor

Returns:

Object:

Example:

try {
    var storage = App.di.createInstance("data", ["compress", "websql"]) ;
}
catch(e) { // will fail if contract does not exist
    console.log(e.name + ': ' + e.message) ;
}

register

  • contract
  • class
  • [dependencies]
  • [options]
)
chainable

Defined in src/di.js:29

Register a class by creating a contract. Use DI/getInstance:method to obtain an instance from this contract/class. The injected dependencies, if any, will be used as constructor parameter in the order provided by the dependencies array.

Parameters:

  • contract String

    name of the contracta

  • class Class

    the class bind to this contract

  • [dependencies] Array optional

    list of contracts; dependencies of this class

  • [options] Object optional

    addition setting used to create the instance.

    • singleton String

      set to TRUE if the class should be treated as a singleton class

Example:

    App.di.registerType("ajax", App.AJAX) ;
    App.di.registerType("ajax", App.AJAX, [], { singleton: true }) ;
    App.di.registerType("util", App.Util, ["compress", "wsql"], { singleton: true } ) ;
