Defined in src/eventhub.js:34

Register a callback to a specific event. Callbacks are executed in the order of registration. Use the 'position' parameter to change this behavior. Note that "before" and "after" are just different stacks, executed before/after the normal execution flow.

Parameters:

  • eventName String
  • callback Function
  • [prepend] Boolean optional

    the callback is placed before all other registered callbacks. They will be executed in this order when the event is trigger

Example:

       Sway.eventHub.on( 'ui.update', this.update.bind(this) ) ;
       Sway.eventHub.on( 'ui.update', this.update.bind(this), true ) ;
