HowTo

To create the information yourself, you must install java and nodeJs and run the following commands: 
$> npm install
$> ./generate-stats
If everything did work, all links below should show interesting stuff!!
  1. Class documentation (yuidoc)
  2. Jasmine unit tests
  3. yuitest code coverage data (JSON)
  4. jscheckstyle output
  5. code coverage
  6. JSPerf performance test
