Create documentation/jscheckstyle/code-coverage

To be able to create the information you must first install nodejs and java.
Next, run the following commands: 
$> npm install
$> ./generate-stats
  1. Class documentation (yuidoc)
  2. Jasmine unit tests
  3. yuitest code coverage data (JSON)
  4. jscheckstyle output
  5. code coverage
  6. JSPerf performance test
